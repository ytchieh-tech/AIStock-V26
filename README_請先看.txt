新增：
1. 手機響應式介面
2. 手機精簡模式
3. K線圖高度可調
4. run_app_8518.bat 使用 0.0.0.0，手機同 Wi-Fi 可直接連線

手機使用：
1. 電腦執行 run_app_8518.bat
2. CMD 顯示 Network URL，例如：
   http://192.168.1.101:8518
3. 手機與電腦連同一個 Wi-Fi
4. 手機 Chrome 打開 Network URL

若要 Android App：
請打開同一個 ZIP 內的 AIStock_V25_Android_WebView_App
用 Android Studio 開啟並打包 APK。
